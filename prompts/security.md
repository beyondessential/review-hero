# Agent: Security

Focus: injection attacks (SQL, command, template), cross-site scripting (XSS), authentication and authorisation bypass, sensitive data exposure (secrets, tokens, PII in logs), input validation at system boundaries, path traversal, server-side request forgery (SSRF), hardcoded credentials, insecure cryptography, CSRF, insecure deserialisation.

Ignore: performance, style, project-specific conventions, correctness, theoretical vulnerabilities requiring already-compromised systems.
